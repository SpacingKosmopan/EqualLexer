## Variable decralation
`$var:7^int;`
// dla nazwy "var" przypisz wartość 7 czyli to liczba całkowita
### Działanie matematyczne
zawsze w `{}`

## Loop declaration
`@i:<2-4)`
// deklaruje i, przechodzi po wartościach dla i: 2, 3 (dla 4 wyłącznie) <br>
`@:x+1$0`
// wykonuj dopóki x jest większe bądź równe zero <br>
`@:x+1$1`
// wykonuj dopóki x jest większe od zera <br>

## Function declaration
`?fun$a^int&$b^int><{a*b}`<br>
`?fun $a^int & $b^int > < { a*b }` <br>
// funkcja zwracająca iloczyn liczb int a i int b <br>
`< ` - // zwróć wartość
