## Variable decralation
`$var:7^int;`
### Działanie matematyczne
zawsze w `{}`
// dla nazwy "var" przypisz wartość 7 czyli to liczba całkowita

## Loop declaration
`@i:<2-4)`
// deklaruje i, przechodzi po wartościach dla i: 2, 3 (dla 4 wyłącznie)
`@:x+1$0`
// wykonuj dopóki x jest większe bądź równe zero
`@:x+1$1`
// wykonuj dopóki x jest większe od zera

## Function declaration
`?fun$a^int&$b^int><{a*b}`
`?fun $a^int & $b^int > < { a*b }`
// funkcja zwracająca iloczyn liczb int a i int b
`< ` - // zwróć wartość
